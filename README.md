# RacketQuickSort
A short QuickSort library made in Racket

## Instructions
Import by dragging RacketQuickSort.rkt into the same folder as your project and putting
```racket
(require "RacketQuickSort.rkt")
```

Usage:
```racket
(quickSort vec)
   vec : vector?
```

The quicksort is a regular single-pivot one.
